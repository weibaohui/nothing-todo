# NothingTodo

A cross-platform CLI tool for managing your daily tasks, built with Rust.

## Preview

![Dashboard](https://raw.githubusercontent.com/weibaohui/nothing-todo/main/docs/dashboard.png)

![Task Detail](https://raw.githubusercontent.com/weibaohui/nothing-todo/main/docs/detail.png)

## Installation

```bash
npm install @weibaohui/nothing-todo
```

Or install globally:

```bash
npm install -g @weibaohui/nothing-todo
```

## Usage

After installation, the `ntd` command will be available:

```bash
ntd --help
```

## Features

- Cross-platform support (macOS, Linux, Windows)
- Fast and lightweight Rust backend
- AI-powered task management with Claude Code and JoinAI executors
- Web UI for visual task management

## Platform Support

- macOS (arm64)
- Linux (x64, arm64)
- Windows (x64)
